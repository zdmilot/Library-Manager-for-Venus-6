Privacy Policy - Library Manager

Effective date: March 1, 2026
Last updated: March 15, 2026

Library Manager ("the App") is a desktop application for managing Hamilton VENUS library packages and related files. This policy explains what information the App processes, where it is stored, and when information may leave your device.

1) Summary

The App is designed to work offline.

The App does not collect telemetry, analytics, advertising identifiers, or usage tracking.

The App does not transmit your library content, settings, or audit data to the developer.

The App includes a "Report issue" link that opens a public GitHub issue page in your default web browser. Any information you submit on GitHub is controlled by you and handled by GitHub.

2) Information the App processes

The App processes information you choose to work with, such as:

Library package files and related metadata

Library files, demo method files, and other workflow files you open or manage

App settings and configuration

Local audit/history records created by the App (if enabled)

This processing occurs on your device to provide packaging, installation, validation, and management functionality.

3) Local-only storage

The App stores operational data locally on your device, which may include:

Installed/known library records and package inventory

Settings and preferences

Local package cache and archives (if you choose to keep them)

Audit trail/history records (if enabled)

No operational data is uploaded by the App. Data remains on your device unless you export it, copy it, or choose to share it outside the App.

4) Network use and data transmission

The App does not require a cloud account and does not include built-in analytics, advertising SDKs, or telemetry collection services.

The App does not transmit your data to the developer's servers.
The App may access the internet only if you choose actions that inherently involve your browser or external services, such as opening a web page link.

4a) Library Store

The App includes an optional Library Store feature that retrieves a package catalog from a public GitHub repository (https://github.com/zdmilot/Library-Manager-Packages). When you open the Library Store, the App downloads a catalog index file and, if you choose to install a package, the package file itself. These downloads are made over HTTPS directly from GitHub's servers. No personal data is sent to GitHub in catalog or package download requests beyond the standard HTTP headers (IP address, user-agent string) that GitHub processes under its own privacy practices.

4b) Ratings and reviews

The App includes an optional ratings and reviews feature that allows you to rate and review library packages. This feature uses GitHub Discussions on the Library-Manager-Packages repository, accessed through a GitHub App integration.

When you submit a review, the following information is posted publicly to a GitHub Discussion:

- Your Windows username (as returned by your operating system)
- Your star rating (1-5)
- Your review comment text (if you enter one)
- Your Windows version (e.g., "Windows 10 Pro 22H2")
- Your installed VENUS software version (if detected)
- The version of the library you are reviewing (if installed)
- The version of the Library Manager app

This information is transmitted to GitHub via the GitHub API and is publicly visible on the GitHub Discussion page. The App does not store this information itself; it is stored by GitHub under GitHub's terms and privacy practices.

Submitting a review is entirely optional. If you do not submit a review, no personal information is sent.

When you view ratings and reviews, the App retrieves publicly posted review data from GitHub Discussions. No personal data is sent in these read requests beyond standard HTTP headers.

Note: Your Windows username is included so that reviews are attributed. If you prefer not to share your Windows username publicly, do not submit reviews.

5) "Report issue" link (GitHub)

The App may provide a "Report issue" option that opens your default web browser to a GitHub page where you can submit an issue.

The App does not automatically send device details (such as Windows version) or application details (such as VENUS version) to GitHub through the issue reporting link.

Any information you enter into the GitHub issue form is submitted by you and may be publicly visible because the repository is public.

GitHub processes information you submit according to GitHub's privacy practices.

Repository: https://github.com/zdmilot/Library-Manager

Recommendation: Do not include sensitive information (such as usernames, machine names, domain names, file paths, license keys, regulated data, or proprietary method/library contents) in public issue reports or reviews.

6) Device, file system, and privilege access

To perform library management workflows, the App may access local file paths and system locations you direct it to use.

Certain operations (for example, installing to protected locations or interacting with COM components) may require administrator approval depending on your configuration and the actions you choose.

7) Data retention and deletion

Local data remains on your device until you remove it, uninstall the App, or use any cleanup options you choose (including uninstaller options, if provided). The developer does not have access to your local data.

8) Security

The App may include integrity, hashing, and signature verification features for package workflows. You remain responsible for endpoint security, OS access controls, and backups in your environment.

9) Children

The App is intended for professional/enterprise use and is not directed to children.

10) Changes to this policy

We may update this policy from time to time. The latest version will be provided in the Store listing and/or in product documentation.
