Regulated Environment
When enabled, only users in authorized Windows security groups (Lab Method Programmer, Lab Service) can perform library management actions such as import, delete, and rollback. Users not assigned to any recognized group will be denied access. This setting can only be changed by authorized users.
Audit Trail Enforcement
Actions are always logged in history, but these options ensure comments and signatures are included for audit trail accountability.
Display
Automatically match the operating system's light or dark appearance. When disabled, a day/night toggle icon is shown in the toolbar and the preference is retained between sessions.
System libraries with warnings or errors will still be shown.
External links to source repositories. This setting is automatically disabled in regulated environment mode.
Data Location
All application data (settings, library registry, audit trail, package backups, exports) is stored in the local/ directory within the application's install folder. This location is globally accessible to all users on this PC.
Unsigned Libraries
Scans the Library folder for .hsl and .smt files that are not part of a signed package or system library. Files and folders starting with ~ or . are automatically excluded (temporary/hidden files). These can be edited and quickly exported as .hxlibpkg packages.
