Library Manager for Venus 6 » Installation

Overview

Library Manager for Venus 6 is distributed as a single Windows installer (LibraryManagerForVenus6_v<version>_Setup.exe) built with Inno Setup. The installer requires administrator privileges and targets 64-bit Windows 10 or later.

System Requirements

RequirementDetails
Operating SystemWindows 10 or later (64-bit)
PrivilegesAdministrator rights required for installation
Disk SpaceApproximately 200 MB (varies with library content)
Default Install PathC:\Program Files (x86)\Library Manager for Venus 6

Installation Wizard Steps

The installer walks you through the following pages in order:

1. Welcome Page

Displays the application name and version. Click Next to proceed.

2. Terms of Use & Privacy Policy

The full text of the Terms of Use and Privacy Policy is presented in a scrollable text area. You must check the “I accept the Terms of Use and Privacy Policy” checkbox before the Next button becomes enabled. Installation cannot proceed without acceptance.

3. Select Destination Directory

Choose the installation folder. The default location is:

C:\Program Files (x86)\Library Manager for Venus 6

You may change this to any writable directory. The installer will create the folder if it does not exist.

4. Application Configuration

This custom page lets you set initial preferences that are written into the application’s settings files on install. The page is divided into three sections:

Environment Mode

OptionDefaultDescription
Enable Regulated Environment Mode (GxP) Off Turns on strict access controls designed for GxP-regulated laboratories. When enabled:
  • Only users in authorized Windows groups (Lab Method Programmer, Lab Service) or Administrators can manage libraries
  • GitHub repository links are disabled and cannot be re-enabled
  • Unsigned libraries are not permitted — all packages must be signed
  • A full audit log is maintained for all operations
  • Import/export operations require authorized group membership
  • Action comments and signatures may be enforced
A confirmation dialog is shown before enabling. See Regulated Environment Mode for details.

Appearance

OptionDefaultDescription
Always use dark mode Off When checked, the application always launches in dark (night) mode. When unchecked, the theme follows the system setting.

GitHub Integration

OptionDefaultDescription
Show GitHub Repository Links Off When checked, GitHub repository links are displayed in library detail views. This option is automatically disabled and locked when Regulated Environment Mode is enabled.

5. Additional Tasks

Optionally create a desktop shortcut for Library Manager for Venus 6. This is unchecked by default.

6. Ready to Install

Displays a summary of all selected options — install directory, Start Menu group, and the configuration choices made on the Application Configuration page. If Regulated Environment Mode is enabled, a prominent notice lists the consequences. Review the summary and click Install to begin.

7. Installation Progress

Files are copied to the destination directory. The following post-install steps happen automatically:

  • Settings files are generated from your configuration choices and written to both {app}\local\settings.json and {app}\db\settings.json.
  • Folder permissions are set on the local\ data directory. The installer runs icacls to grant Modify permissions to the Users group so that non-administrator user accounts can read and write application data.

8. Completion

Once installation finishes you may choose to launch Library Manager for Venus 6 immediately.

Start Menu & Desktop Shortcuts

The installer creates a Start Menu group containing:

  • Library Manager for Venus 6 — launches the application
  • Uninstall Library Manager for Venus 6 — runs the uninstaller

A desktop shortcut is created only if the corresponding checkbox was selected during installation.

Silent / Unattended Installation

The installer supports Inno Setup’s standard silent-install switches:

LibraryManagerForVenus6_v1.4.8_Setup.exe /SILENT
LibraryManagerForVenus6_v1.4.8_Setup.exe /VERYSILENT /SUPPRESSMSGBOXES

In silent mode, default configuration values are used (Regulated Mode off, system theme, GitHub links hidden). To create a desktop shortcut in silent mode, add /TASKS="desktopicon".

Upgrading from a Previous Version

When reinstalling or upgrading, the installer overwrites application binaries and template files but preserves user data in the local\ directory. On first launch after an upgrade, the application automatically migrates data from any legacy storage locations. See File Storage & Data Management for migration details.
