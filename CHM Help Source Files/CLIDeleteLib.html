CLI: delete-lib
Library Manager for Venus 6 » CLI Reference » delete-lib

Description

Removes library files from disk and updates the database record. By default, performs a soft delete that preserves the record with a deleted=true flag for history. Requires explicit --yes confirmation to prevent accidental deletion.

Usage

node cli.js delete-lib (--name <name> | --id <id>) --yes [options]

Options

FlagRequiredDescription
--name <name>Required*Library name. *Use either --name or --id.
--id <id>Required*Library database ID. *Use either --name or --id.
--yes / --force / -yRequiredExplicit confirmation that deletion should proceed
--hardNoPermanently remove the database record (no history retained)
--keep-filesNoRemove from database only; leave installed files on disk
--db-path <dir>NoOverride the user data directory path

Delete Process

  1. The library record is looked up by name or ID
  2. System library check - system libraries return SYSTEM_LIBRARY_READ_ONLY error
  3. Unless --keep-files is set:
    • All library files and help files are deleted from the library install path
    • All demo method files are deleted from the demo install path
    • Empty directories are removed after file deletion
  4. Database record is updated:
    • Soft delete (default): deleted=true and deleted_date are set
    • Hard delete (--hard): record is permanently removed
  5. The library is removed from any group assignments in the navigation tree

Examples

:: Soft-delete (history preserved)
node cli.js delete-lib --name "MyLibrary" --yes

:: Hard delete (record removed entirely)
node cli.js delete-lib --name "MyLibrary" --yes --hard

:: Remove from database but leave disk files
node cli.js delete-lib --name "MyLibrary" --yes --keep-files

:: Delete by database ID
node cli.js delete-lib --id a1b2c3d4 --yes

Warning: COM DLLs are not automatically deregistered by the CLI. If the library had COM-registered DLLs, run the 32-bit RegAsm.exe with elevated privileges manually:

C:\Windows\Microsoft.NET\Framework\v4.0.30319\RegAsm.exe /unregister <dll>

Do NOT use Framework64 — VENUS is a 32-bit application. See COM DLL Registration for details.
