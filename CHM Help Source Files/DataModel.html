Overview

Venus Library Manager uses diskdb, a lightweight JSON-backed database, to persist all application state. The database resides in the db/ directory within the application folder. Each collection is stored as a separate .json file.

installed_libs Collection

Each record represents one installed library. This is the primary data store for the application.

file_hashes Object

Maps each integrity-tracked file to its SHA-256 hash. Only files with extensions .hsl, .hs_, .sub, and COM .dll files are tracked.

"file_hashes": {
  "ExampleLib.hsl": "612b778dde75f7e93c2894845e1f9f7e...",
  "ExampleLib.hs_": "cdd4d99218e02df8972ab8dbb1b6be43...",
  "ExampleLib.dll": "f30cb612945c0af795b134cd6b964e70..."
}

public_functions Array

Each entry describes a public function parsed from the library's .hsl files:

{
  "name": "Add",
  "qualifiedName": "ExampleComHelper::Add",
  "params": [
    { "type": "variable", "name": "a", "byRef": false, "array": false },
    { "type": "variable", "name": "b", "byRef": false, "array": false }
  ],
  "returnType": "variable",
  "doc": "Function documentation from comments...",
  "file": "ExampleComHelper.hsl"
}

links Collection

Stores VENUS tool shortcuts and directory shortcuts displayed in the GUI sidebar. Records come in two types:

Tool Shortcut (type: "file")

{
  "_id": "met-editor",
  "name": "Method Editor",
  "description": "",
  "icon-customImage": "HxMet.png",
  "icon-class": "fa-folder",
  "icon-color": "color-blue",
  "path": "C:\\Program Files (x86)\\Hamilton\\Bin\\HxMetEd.exe",
  "type": "file",
  "default": true,
  "favorite": true,
  "last-started": "",
  "last-startedUTC": 0
}

Directory Shortcut (type: "folder")

{
  "_id": "lib-folder",
  "name": "Library",
  "description": "VENUS software library files",
  "icon-customImage": "",
  "icon-class": "fa-folder",
  "icon-color": "color-blue",
  "path": "C:\\Program Files (x86)\\Hamilton\\Library",
  "type": "folder",
  "default": true,
  "favorite": true
}

groups Collection

Defines the navigation groups (tabs) shown in the GUI. Each group has a name, FontAwesome icon class, and display metadata.

{
  "_id": "gAll",
  "name": "All",
  "icon-class": "fa-th",
  "default": true,
  "navbar": "left",
  "favorite": true
}

Default Groups

IDNameNavbar SidePurpose
gAllAllleftShows all installed libraries
gRecentRecentleftRecently accessed libraries
gFoldersImportrightImport workflow tab
gEditorsExportrightExport workflow tab
gHistoryHistoryrightAction history

User-created groups have auto-generated _id values and "default": false.

tree Collection

Maps groups to their member libraries. Each record links a group-id to an array of library _id values.

{
  "_id": "0fd24a893e6544688b33102aa00739f3",
  "group-id": "7d23d70f46ee4071adcfaffbb99f4f47",
  "method-ids": [
    "6398df8b4d2e4555b44b719910f3db60",
    "73519ab128354b2bace3eb37ca237d20"
  ],
  "locked": false
}
FieldDescription
_idTree record ID
group-idThe group this membership record belongs to
method-idsArray of library _id values assigned to this group
lockedWhether the membership is locked against editing

settings Collection

Stores application preferences. Always a single record with _id of "0".

{
  "_id": "0",
  "recent-max": "20",
  "chk_confirmBeforeInstall": true,
  "chk_overwriteWithoutAsking": false,
  "chk_autoAddToGroup": true,
  "userDataPath": "C:\\Program Files (x86)\\Hamilton\\Library\\.VenusLibraryManager",
  "windowMaximized": true
}
FieldTypeDescription
recent-maxstring (number)Maximum number of items in the Recent group
chk_confirmBeforeInstallbooleanPrompt for confirmation before importing
chk_overwriteWithoutAskingbooleanSkip overwrite confirmation for existing libraries
chk_autoAddToGroupbooleanAutomatically assign imported libraries to a user-created group
userDataPathstringUser data directory (where the user's copy of the database is stored)
windowMaximizedbooleanWhether the main window should start maximized

User Data Directory

The application database in the db/ folder serves as the default template. On first run or when importing, the user's working database is stored at the configured userDataPath:

C:\Program Files (x86)\Hamilton\Library\.VenusLibraryManager\

This directory mirrors the db/ structure and contains the live, mutable copy of the database that reflects the current installation state.
