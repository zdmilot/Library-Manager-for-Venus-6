Venus Library Manager

Overview

Venus Library Manager is a desktop application and command-line tool for managing Hamilton VENUS libraries and related assets. Built on NW.js and Node.js, it provides a complete package lifecycle for .hxlibpkg (single library package) and .hxlibarch (multi-library archive) files.

Venus Library Manager enables laboratory automation engineers to create, import, export, update, roll back, and verify Hamilton VENUS HSL libraries through both a graphical user interface and a scriptable CLI. It replaces the error-prone manual process of copying library files into the VENUS directory tree with a structured, versioned, integrity-checked workflow.

Key Capabilities

  • Visual Library Management — Browse installed libraries as cards with icons, metadata, version, author, tags, and integrity status indicators
  • Package Import — Import single .hxlibpkg packages or multi-library .hxlibarch archives with pre-install previews and validation
  • Package Export — Export installed libraries back to portable package files, capturing their current on-disk state (mutable export)
  • Package Creation — Build .hxlibpkg files from raw HSL source files and metadata using the GUI Library Packager or the create-package CLI command
  • Library Deletion — Safely remove libraries with COM-aware workflow, soft-delete history, and optional hard-delete
  • Version Rollback — Automatically cache every imported package and roll back to any previously installed version on demand
  • File Integrity Verification — SHA-256 hashing for user libraries and Hamilton metadata footer verification for system libraries
  • Library Grouping — Organize libraries into custom groups with drag-and-drop reordering and assignment
  • VENUS Tool Integration — Launch Hamilton editors (Method, Liquid Class, Labware, HSL, System Config) directly from the application
  • Command-Line Interface — Full-featured CLI (cli.js) for scripted automation, CI/CD integration, and headless operation

System Requirements

  • Operating System: Microsoft Windows 10 or Windows 11
  • VENUS Version: 4.7 and above (recommended: 6.x)
  • Runtime: Node.js (for CLI usage independent of the GUI)
  • Default Library Path: C:\Program Files (x86)\HAMILTON\Library
  • Default Methods Path: C:\Program Files (x86)\HAMILTON\Methods
