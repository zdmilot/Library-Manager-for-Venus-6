CLI: list-publishers
Library Manager » CLI Reference » list-publishers

Description

Lists all registered publisher certificates in the local publisher registry, showing their key ID, fingerprint, organization, and creation date.

Usage

node cli.js list-publishers [--json]

Options

FlagRequiredDescription
--jsonOptionalOutput raw JSON for scripting/CI integration

Output

For each registered certificate, the following information is displayed:

  • Publisher name
  • Organization (if set)
  • Key ID — first 16 characters of the fingerprint (short identifier)
  • Fingerprint — full SHA-256 hex of the Ed25519 public key
  • Created date — ISO 8601 timestamp of certificate generation

Examples

:: List all publishers (human-readable)
node cli.js list-publishers

:: List as JSON for scripting
node cli.js list-publishers --json
